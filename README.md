# glyphormat
Simple formatter for .glyphs files

```
usage: glyphormat.py [-h] [-o FILE | -] [-s SPACES | -t] inputfile

Format .glyphs files

positional arguments:
  inputfile    the input file to process

optional arguments:
  -h, --help   show this help message and exit
  -o FILE | -  where to write the output, use - for STDOUT
  -s SPACES    use SPACES number of spaces for indentation
  -t           use tabs for indentation
```
